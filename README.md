# Sistema de Avaliação Escolar (Python)

Projeto de simulação de avaliação e classificação de alunos.

## Funcionalidades
- Recebe notas dos alunos
- Calcula média
- Classifica alunos como Aprovado ou Reprovado
- Mostra resultados claros no terminal

## Como rodar
1. Abra o terminal
2. Execute: `python main.py`
